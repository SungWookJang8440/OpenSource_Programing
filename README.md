## 기능
총 3개의 기능(페이지)으로 구성되어 있습니다.

1. **`/` (Home)**: API 서버 접속 환영 메시지 출력
2. **`/profile` (Profile)**: 개발자(장성욱 - 경북대학교 컴퓨터학부)의 기본 인적 사항 출력
3. **`/skills` (Skills)**: 주력 개발 스택(Python) 출력

# 1. 패키지 설치

# 2. 서버 실행
